# learning-spring-security

This is a project to learn Spring Security from the Udemy course Spring Security Zero to Master along with JWT, OAuth2.

This project covers:
* Spring Security to secure a web application
* Configuring authentication and authorization
* Implementing role based access using roles and authorities
* How to handle common attacks like CORS and CSRF
* JWT and its role in authentication and authorisation
* Using OAuth2, OpenID to secure a web application
* Authorization servers like Keycloak, Spring Authorization server
